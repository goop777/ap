Banshee Patch Notes

2026-04-27

NEW UPDATE

general

  • tabs that cant be seen due to screen resolution (on laptops mostly) will now go under the first tabs
  • show toggles now disabled by default
  • unloading the script (via settings -> ui settings -> unload script) now disables script execution on serverhop (until you reexecute)

deepwoken & deepwoken farms

  • added loot all to auto loot
  • added strict loot
  • use strict loot (should it loot items in the textbox)
  • only strict loot (should it only loot items in the textbox)
  • added always loot deep gems

deepwoken

  • added flow state mode (has dropdown with attack toggles)
  • added heretic's sutra mode (minimum sanity slider)
  • added golden tongue mode
  • added wisp delay slider
  • added block running m1s (meant for auto sprint users)
  • added no swimming drain (prevents the game from taking thirst/hunger while swimming)
  • added auto weapon manual minigame
  • added trial of 1 teleport
  • added guild door teleport
  • added new ping math, should be more accurate on spikes/high ping (auto parry) thanks to Mac
  • added FOV, wall check and filters for silent aim
  • re-arranged features and renamed things in the UI
  • boat teleportation is now more consistent and you don't have to retoggle to update the location/boat anymore
  • you can now switch back to the default ap config; #script-info
  • the auto parry manager is now more organized and includes proper attack names
  • fixed rare scenario where the game would "break" after respawn
  • fixed no enforcer pull not working if walkspeed or flight enabled
  • fixed silent aim
  • fixed spectate
  • fixed void, more accurate, even better
  • fixed depths esp sometimes not register (oops)
  • fixed various features not working after respawn
  • fixed ap debugger having negative ideal timings

added deepwoken farms

  • seperate script (same loader) that has all the banshee farms in it, change the script type to "farms"
  • all farms from the main script moved to this
  • added authority farm
  • random pick authority talent toggle
  • serverhop on player toggle (smart checks, only when you can be exposed)
  • auto eat w/ threshold slider
  • travel speed slider
  • you can now press F1 to stop farming (will tp to menu and remove script execution on serverhop)
  • titus farm unpatched
  • echoes farm now selects the random attunement to help the user unlock attunements while farming echoes on a fresh account
  • adjusted the player's height during the self dialogue sequence (echo farm)
  • auto moonseye (for auto bank, titus farm) is now a toggle instead of being enabled by default
  • made the custom tween non fps reliant, won't slow down on fps drops anymore (#bug-reports if issues)
  • fixed auto bank (titus) not stopping when out of knowledge
